1.全局安装 npm install-g vue-cli vue示例安装 2.vue init webpack-simple my-project 3.cd my-project 下载第三方模块 4. npm install 5 npm run dev 在package.json中有两种方式 分别是script:{'dev':..........开发环境 和 'build':......生产环境} dev....webpack-dev-server 服务 hot:热更新：只要页面发生改变， 不需要刷新浏览器 它会自动在浏览中自动更新 6.app.vue:单文件组件 main.js入口文件 7.render:h =>h(app)渲染模板的方式跟集成模板类似 ####8.app.vue或者任何.vue单文件都必须在main.js中引入 和new Vue实例化 并渲染模板 当执行npm run dev时候，vue会自动执行webpack打包 并把打包文件放在/dist/文件夹下 出口：build.js 但是在项目中并没有看见dist文件夹 是因为 webpack-dev-server会自动删除该文件夹
